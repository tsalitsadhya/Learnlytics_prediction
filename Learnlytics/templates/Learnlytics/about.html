About Us

This project is developed by a team of 5 Data Science Bootcamp participants. We created a web-based application to visualize and test machine learning models.

Embun Aqeela Zahra

Use Case: Learning Style Analysis

Contribution: Developed a web-based analytics dashboard to classify student learning types using K-Means clustering, enabling the identification of "Intensive", "Relaxed", and "Passive" learners. The system includes dynamic visualizations such as distribution charts and activity heatmaps, along with a detailed student profile table showing learning metrics (average duration, session frequency, and night activity). It supports interactive filtering and provides actionable insights for learning behavior analysis.

Kezia Joseph

Use Case: Student Grade Prediction

Contribution: Developed a machine learning-based prediction system using Random Forest Regressor to estimate student grades from activity log features. The system is integrated into a Django web application, with automated model training via custom management commands and persistent model storage using joblib. Data is sourced from a structured ETL pipeline, and the output supports early identification of at-risk students. The model achieves measurable performance metrics (MAE and R²) and enables scalable academic insight generation for educators.

Raffila Shalwa Sakina

Use Case: Study Partner Recommendations

Contribution: Developed data processing and Apriori-based association rule mining using courses and grades to generate study partner recommendations. Integrated the engine into a Django app with a user-friendly form and real-time scoring. Created interactive visualizations of association rules and designed the UI through team collaboration.

Riska Atikah Rahmawati

Use Case: Course Predictions

Contribution: Contributed to data collection, cleaning, and feature selection to build a course recommendation prediction model for students using Random Forest Classification. The model was integrated into a web application through a Django API to provide real-time recommendations, accompanied by interactive visualizations using Chart.js. Led the team in project development, managed the timeline, designed the user interface (UI) based on team discussions, and conducted regular model retraining to improve the accuracy and relevance of the predictions.

Tsalitsatul Adhya

Use Case: Student Graduation Predictions

Contribution: Developed a predictive model to estimate whether a student would pass or fail a specific course, based on behavioral data such as activity duration and frequency. The model enables early identification of students who may require intervention. Contributed to data preprocessing, handled system errors during development and deployment, and supported the team in improving system robustness.
